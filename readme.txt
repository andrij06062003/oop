 Створити клас – матриця, який у закритій частині містить двовимірний масив для зберігання її елементів, кількість рядків та стовпців. Визначити конструктор, деструктор, 
 функції для зміни та виведення елементів матриці, пошуку  мінімального та максимального елементів, обчислення суми елементів матриці.
